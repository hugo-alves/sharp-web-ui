# SHARP - Single-Image 3D Gaussian Splatting

Generate 3D Gaussian Splats from a single image in under a second.

## Features

- **Fast**: Sub-second inference on GPU
- **Simple**: Just upload an image
- **Compatible**: Output works with any 3DGS viewer

## Usage

1. Upload an image
2. Click "Generate 3D Gaussians"
3. View the 3D preview or download the PLY file
